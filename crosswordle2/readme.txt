Welcome to Cross Wordle

This game was created for the Future Business Leaders of America
Computer Game and Simulation Programming.

To run the game, either go to the link:
https://matteosalverio.github.io/ratlab/projects/crosswordle2/
or open the index.html file in this folder. This will open a
webpage containing the game.

All visual and UI code can be found in the index.html and styles.css
files, and all game code can be found in the script.js file.
