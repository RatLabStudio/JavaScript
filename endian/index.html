About

This game was designed to be a recreation of a game called 0x10c, created by Markus Persson, cancelled in 2013. The game I made features multiplayer, an in-game computer onboard the ship, guns, health, boxes, and other things. The game is clearly unfinished, lacking player models, animations, and actual gameplay. It is more of a functional concept at the moment.

Future Plans for Endian

I do plan to continue working on this game in some capacity. The issue is, it is a massive project to work on and takes a lot of skill to do things properly. If I am going to release a game like this, it has to be at least decent. I don’t know what this game will look like in the future, or if it will even keep the name, but I do hope to expand off of it at some point.
